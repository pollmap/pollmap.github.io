# pollmap.github.io

채용자가 3분 안에 핵심 산출물을 이해하도록 정리한 개인 포트폴리오 허브입니다.

## Positioning

> 금융 데이터·AI·리서치 자동화를 연결하는 학부생 빌더

이 페이지는 창업가/헤지펀드/검증되지 않은 수치 중심이 아니라, 취업 포트폴리오 관점에서 다음 증거를 보여줍니다.

- KOFIA 채용시장 데이터 분석
- WM/PB 브리핑 자동화 MVP
- 금융 리서치/데이터 자동화 레포
- 자산평가·운용지원·증권 데이터·WM 리서치 직무 타깃

## Stack

- Static HTML/CSS/JS
- GitHub Pages
- `projects.json` 기반 프로젝트 카드

## Local Preview

```bash
python -m http.server 8000
```

Open `http://localhost:8000/`.

## Editing Projects

Update `projects.json`. Only the first three `featured: true` projects are shown in the main grid.
